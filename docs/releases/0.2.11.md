# Workspace 0.2.11

Workspace 0.2.11 is a focused desktop lifecycle patch for the native macOS Quit command introduced in 0.2.10.

## macOS Quit behavior

Choosing **Workspace → Quit Workspace** or pressing **Command-Q** now performs the bounded Workspace shutdown once and terminates the application on the first request. The app closes its local API, flushes Pi state, stops restricted-app hosts, disposes the updater, and then resumes Electron termination on a fresh event-loop turn. The window and running indicator leave the Dock together; a second **Dock → Quit** action is no longer necessary.

Dock, system, extension-requested, CLI, Windows menu, and updater-driven exits share the same serialized coordinator. Concurrent quit requests cannot duplicate cleanup. A downloaded updater retains its dedicated `quitAndInstall` handoff, and an unexpected cleanup error is reported without stranding an invisible process.

## Why this patch was needed

Version 0.2.10 used Electron's native macOS `quit` role and cancelled its first `before-quit` event while asynchronous host cleanup ran. The continuation could call `app.quit()` before the native menu termination cycle had fully unwound. That reentrant request was swallowed: the window closed, macOS correctly kept the application process alive, and a later Dock quit succeeded because cleanup had already completed.

The application-menu item now enters Workspace's graceful coordinator directly, keeps the standard **Command-Q** accelerator, and defers only the final Electron quit. Automated coverage locks the deferred first-request behavior, updater handoff, cleanup failure fallback, pre-authorized exits, and native-menu wiring.

## Upgrade notes

Windows and macOS continue to share version 0.2.11 on their separate public update feeds. Existing Mac installs can choose **Workspace → Check for Updates…**; the signed Squirrel.Mac payload replaces the application and relaunches it. Spaces, Chats, History, preferences, Pi state, restricted-app data and grants, and ordinary files remain in their existing locations.

The Mac release remains Apple silicon only and requires macOS 12 or newer. All macOS signing, notarization, Gatekeeper, remote-digest, installed-updater, application-menu Quit, Command-Q, Dock Quit, and close/reopen gates must pass before this patch is considered complete.
